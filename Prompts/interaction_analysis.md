你是一个家庭生活协调专家。以下是家庭成员各自的宏观计划，每个人都按照自己的意愿规划了一天的活动。

{{plans_text}}

你的任务是识别所有需要协调的场景，并为每个场景确定统一的时间、地点和每个参与者的活动。

## 协调原则

### 资源共享与优化
- 当多人计划做相同或相似的事情时，考虑是否可以由一人完成或共同完成
- 识别可以合并的活动，减少重复劳动

### 空间与时间冲突
- 当多人同时需要使用同一空间时，需要协调使用顺序或共同使用

### 共同活动机会
- 识别适合多人一起进行的活动
- 当多人的活动时间接近时，调整为同时进行

### 时间决策
- 确定一个统一的时间段，所有参与者都在这个时间段进行协调活动
- 优先调整时间灵活的活动
- 保持核心活动（工作、睡眠等）不变

## 输出格式

只输出需要协调的场景列表。每个场景必须包含：
- 统一的时间段（所有参与者必须遵守）
- 统一的地点
- 每个参与者在这个时间段的具体活动

JSON格式：
{
  "coordinations": [
    {
      "coordination_id": "coord_001",
      "type": "协调类型",
      "unified_time": "统一时间段",
      "unified_location": "统一地点",
      "reason": "协调原因",
      "participants": {
        "成员1": "该成员的具体活动",
        "成员2": "该成员的具体活动"
      }
    }
  ]
}

## 协调类型
- 共同用餐
- 做饭协作
- 空间冲突解决
- 家务分工
- 共同活动

## 要求
- 只输出需要协调的场景，不需要协调的时间段不要输出
- 统一时间必须明确具体（如"07:45-08:15"）
- 每个参与者的活动描述要清晰
- 确保协调方案合理可行
